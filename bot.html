Cooper Discord Bot

Invite Cooper to your Discord server to greet new members and help with simple server structures.

Discord Bot

General features that Cooper offers.

Welcome Message

When someone joins your server Cooper will post a configurable welcome message to your chosen channel.

Channel Requirements

To receive welcome messages, create a text channel whose name includes the word welcome (for example: #welcome, #server-welcome).

Role Requirement

Cooper expects a server role that includes the word Member (for example: Member, Server Member) to assign or reference for basic onboarding.

Functions

Commands that Cooper can run.

/rules

Posts the server rules into a channel whose name includes rules. Restricted to users with a role that includes Developer.
